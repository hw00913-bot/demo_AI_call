# 业务规则

## 标签排序移位规则

修改标签排序值时，系统自动调整同标签池（供应商模式）或同标签集（中台模式）内其他标签的排序，保持排序值连续不重复。

- **后移**（newSort > oldSort）：排序值在 `(oldSort, newSort]` 区间内的其他标签，sort 减 1
- **前移**（newSort < oldSort）：排序值在 `[newSort, oldSort)` 区间内的其他标签，sort 加 1
- **不变**（newSort = oldSort）：无操作

示例：A=1, B=2, C=3, D=4，编辑 C 的排序：
- C: 3→1 → A=2, B=3, C=1, D=4
- C: 3→4 → A=1, B=2, C=4, D=3

## 标签启用自动映射

供应商标签首次启用时，系统根据标签名称关键词自动推断映射到本地标准标签：
- 高/强烈/S级/A级/近期成交 → 高意向/高优先级类
- 中/B级/C级/考虑/潜在 → 中意向/需跟进类
- 低/D级/E级/一般 → 低意向/门店处理类
- 无效/空号/停机/异常 → 无效客户/服务预警类

## 停用保护

供应商或场景状态为 `disabled` 时，对应配置节点下的启用勾选、编辑、删除、新增操作均禁用。

## 权限范围

标签管理模块所有功能仅限超级管理员使用。搜索和状态筛选属于前端过滤，不涉及权限控制。
