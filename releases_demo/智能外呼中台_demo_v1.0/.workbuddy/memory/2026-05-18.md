# 2026-05-18 工作记录

## 智能外呼中台 - 架构重构与新增外呼列表页

### 已完成

1. **SPA-Lite 架构重构**
   - 将单文件 monolith（index.html + script.js）拆分为模块化架构
   - 新目录结构：
     - `data/mock.js` — 模拟数据集中管理（12条外呼任务 + 线索统计NEV/ICE + 线索明细 + 回流）
     - `js/common.js` — 公共工具（toast、导出、多选下拉、级联菜单、Tab切换、日期初始化）
     - `js/pages/report-clue.js` — 线索统计页完整迁移（3个主Tab + 2个子Tab + 回流统计）
     - `js/pages/scene-list.js` — 外呼列表卡片页（新开发，基于截图还原）
     - `js/nav.js` — 路由导航 + 面包屑 + 菜单交互
   - 旧 `script.js` 已删除

2. **外呼列表页（scene-list）**
   - 基于用户提供的截图实现，包含：
     - 白色页面标题区：「外呼列表」+ 副标题说明
     - 筛选栏：场景名称输入框 + 状态下拉 + 重置/查询按钮
     - 4列卡片网格（响应式：大屏4列、中屏3列、小屏2列、移动端1列）
     - 每张卡片：标题行（含状态圆点）+ 来源标签 + 三指标统计（已分配/待呼叫/已呼叫）+ 操作链接
     - 三种状态：已完成（绿色）、等待重呼（红色）、外呼中（橙色）
     - 两种来源标签：手动导入（蓝色系）、接口传入（绿色系）

3. **面包屑导航**
   - 顶部栏中央动态面包屑：智能外呼 / 外呼场景 / 外呼列表
   - 点击可跳转，支持一级面包屑返回

4. **占位页面**
   - 所有未开发模块自动显示「功能开发中」占位页（10个模块）
   - 包括：首页、通话统计、计费统计、外呼拦截、通话记录、线索记录、账号/租户/通道/业务场景管理

### 技术决策
- 采用零构建工具的 SPA-Lite 模式，index.html 作为 Shell，JS 模块通过 innerHTML 动态渲染
- 每个 page 模块导出 `{ render(), init() }` 统一接口
- 样式保持单文件 style.css，通过 BEM-like 命名隔离各页面样式
