# 智能外呼中台 - 长期记忆

## 项目概述

智能外呼中台，自建平台，对接第三方智能平台完成对手机号的智能外呼。

**已对接第三方平台：**
- 冰兰
- 一知科技
- 科大讯飞

---

## 账户权限模型

| 角色 | 数据可见范围 | 操作权限 |
|---|---|---|
| **超级管理员** | 所有租户数据 | 只能操作自己创建的任务，不能操作非超管创建的任务 |
| **租户管理员** | 本租户数据 | 可查询和操作本租户创建的任务和数据 |
| **租户运营** | 本租户数据 | 仅查看，无操作权限 |
