### 步骤跳转 Bug 修复（抽屉改造遗留问题）
- **根因**：`goToStep()` 内部第411/413行仍引用未定义的 `modal` 变量（应为 `drawer`），导致点击"确定"时 JS 抛出 `ReferenceError`，步骤完全无法切换
- **修复**：两处 `modal` → `drawer`
- **教训**：全局搜索替换选择器名时需同步检查函数内部所有变量引用，不能只改函数头部的查询选择器
